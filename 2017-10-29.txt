安装Apache
# yum -y install httpd
添加linux用户&设置密码：
# adduser -d /var/www/html siteadmin
# passwd siteadmin
修改目录权限：
#chown -Rf siteadmin.siteadmin /var/www/html
启动Apache:
# service httpd start
